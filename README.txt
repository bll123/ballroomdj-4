Release Notes:

  4.4.1:
    Linux Installation:
      Run the linux-pre-install-v9.sh script.
    MacOS Installation:
      Run the macos-pre-install-macports-v16.sh script.

  4.3.3.6:
    Please upgrade to version 4.3.3.6 if you are using
      version 4.3.3.1 through 4.3.3.5.
    Linux Installation:
      Run the linux-pre-install-v8.sh script.
    MacOS Installation:
      Run the macos-pre-install-macports-v14.sh script.

  4.3.2.4:
    The Configuration / General option 'BPM' has been set to MPM.
    If you want to display beats per minute, reset this configuration
    item back to BPM.

  Not yet implemented:
    - Auto-Organization.
    - Music Manager:
      - Audio Identification.

  This installation will not affect any BallroomDJ 3 installation.
  (The features that rename audio files is not implemented yet).
  (Writing audio file tags is turned off upon conversion, and the
   'Write BDJ3 Compatible Audio Tags' is on).
