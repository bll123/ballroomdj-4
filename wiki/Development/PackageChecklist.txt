* on Linux:
    * update README.txt
    * update VERSION.txt
    * update ../wiki/en/Change%20Log.txt
    * make distclean
    * make 2>&1 | tee w
    * make check
    * ./utils/dbtest.sh
    * ./utils/insttest.sh
    * make testsuite
    * ../pkg/mkpkg.sh
<br>Recall that any time the mkpkg.sh script is run on Linux, the source
must be re-synced to the other platforms and mkpkg.sh must be run again
on the other platforms.
    * ../dev/rs.sh
* for each macos, windows
    * make distclean
    * make 2>&1 | tee w
    * egrep warnings w
* for each macos, windows
    * make check
    * ./utils/dbtest.sh
    * ./utils/insttest.sh
    * make testsuite
* for each macos, windows
    * ../pkg/mkpkg.sh
* for each linux, macos, windows
    * ../web/push.sh
* on linux
    * hg: check in
    * hg: create a tag for the release
    * push to sourceforge
    * ../web/wikiput.sh putall
* final
    * announce the release on the forum
    * announce the release on freshcode.club
    * update sourceforge, mark the packages
