### Configure : Audio Sources

The topic list may need to be scrolled down to view this configuration
topic.

Audio sources configures access to remote audio servers and to set up
a BDJ4 audio server.  At this time (2025-6-13), BDJ4 servers,
BDJ4 clients and Podcast clients are supported.

Adding a new audio source: Select the __Add&nbsp;New__&nbsp;(![Add
New](https://ballroomdj4.sourceforge.io/wikiimg/AddNewButton-E.png))
button to add a new audio source. The audio source is added with a
default name of _New&nbsp;Audio Source_.  Change the name of the audio
source to what you want.

Removing an audio source: Select the audio source to remove in the
list.  Select the
__Delete__&nbsp;(![Delete](https://ballroomdj4.sourceforge.io/wikiimg/DeleteButton-E.png))
button to remove the audio source.

__Mode__: Set to _Off_, _Client_ or _Server_.

__Name__: Enter a name to identify the audio source.

__Type__: The type of audio source.  At this time (2025-6-13),
_BDJ4_ and _Podcast_ are supported.

__URL__: When configuring a client, enter the URL of the remote
server.  Do not include the port number.

__Port__: When configuring a client or a server, enter the port number
that the client connects to or which the server uses.

__User__: The user ID to authenticate the connection.

__Password__: The password to authenticate the connection.

__Check Connection__:  Once a client is configured, use the
__Check&nbsp;Connection__ button to verify that the client is
configured correctly.  The server must be accessible.
