### Configuration : Mobile Remote Control

<span style="color:#4559bf;">_Mobile Remote Control can be used to
control BDJ4 from your mobile device._</span>

Mobile Remote Control is a web page that can be used to control BDJ4
over the Wi&#8209;Fi network.  Access the web page via your mobile phone or
tablet.  The computer running BDJ4 must be connected to a Wi&#8209;Fi
network, but does not need to be connected to the internet.

___Enable Remote Control___: Turns on the remote control function
(activated when the player is started).

___HTML Template___: Select the style (and language) of web page to
display.

___User ID___: The user ID to use when signing on to the remote control
web page.

___Password___: The password to use when signing on the the remote
control web page.

___Port___: The port to connect to.  This usually does not need to be
changed unless there is another device using this port.

___QR Code___: Select the link to display a QR code in your browser.
The QR code can be scanned with your mobile phone or tablet.
