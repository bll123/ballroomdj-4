### Configure : Edit Status

The status field can be used to track the status of songs in the
database.  If you want to use the status field, you will also want to
change the
[Display&nbsp;Settings](en-Configuration-Display%20Settings).

The _New_ and _Complete_ status may not be removed or have their
names changed.  Songs newly added to the database are assigned the
_New_ status.

The __Play?__ selection indicates whether the song will be used in
automatic or sequenced playlists and whether it will show up in the
request selection.  This selection corresponds to the
_Playable&nbsp;Status_ filter.
<br>__References:__
<br>[Configuration&nbsp;/&nbsp;Dialogs](en-Configuration-Dialogs)

Adding a status: select the __Add&nbsp;New__
(![Add&nbsp;New](https://ballroomdj4.sourceforge.io/wikiimg/AddNewButton-E.png))
button.  Click on the status name to edit the name.  Use the up and
down arrows
(![move-up](https://ballroomdj4.sourceforge.io/wikiimg/UpButton-B.png),
![move-down](https://ballroomdj4.sourceforge.io/wikiimg/DownButton-B.png))
to adjust the order in which the statuses appear.

Removing a status: Select the status and select the __Delete__
(![Delete](https://ballroomdj4.sourceforge.io/wikiimg/DeleteButton-E.png))
button.
