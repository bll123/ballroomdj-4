### Configure : iTunes

#### Configuring iTunes

To allow BallroomDJ&nbsp;4 to access the iTunes information, iTunes
must be configured to share the iTunes library information.  In iTunes,
select the __Edit&nbsp;/&nbsp;Preferences__ menu item.  In the
preferences dialog, select the __Advanced__ tab.

In the iTunes advanced preferences, select the __Share iTunes Library
XML with other applications__ checkbox.

#### Configuring BallroomDJ 4 for iTunes

__iTunes Media Folder__: When the iTunes media folder is selected,
BallroomDJ&nbsp;4 will try to locate the associated iTunes XML file.

__iTunes XML File__: The iTunes XML File location is required.

__Rating__: The rating selections set what rating to assign for the
associated star rating in iTunes when the
[Update&nbsp;from&nbsp;iTunes](en-Management-Update%20Database)
function is used.

__Fields to Import__: Select which fields to import when the
[Update&nbsp;from&nbsp;iTunes](en-Management-Update%20Database)
function is used.
