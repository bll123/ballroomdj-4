### Configuration : Edit Genres

<span style="color:#4559bf;">_Genres are optional._</span>

If you want to use genres, you will also want to change the
[Display&nbsp;Settings](en-Configuration-Display%20Settings).

To add a new genre, select the _Add New_ (![Add
New](https://ballroomdj4.sourceforge.io/wikiimg/AddNewButton-E.png))
button.  Then click on the name to change the name.  Select the
_Delete_
(![Delete](https://ballroomdj4.sourceforge.io/wikiimg/DeleteButton-E.png))
button to delete a genre.

Use the up and down arrows
(![move-up](https://ballroomdj4.sourceforge.io/wikiimg/UpButton-B.png),
![move-down](https://ballroomdj4.sourceforge.io/wikiimg/DownButton-B.png))
to change the order of the genres to match your preference.

The _Classical?_ selection is used for classical music.  __Classical
music support is not yet implemented in BDJ4.__
