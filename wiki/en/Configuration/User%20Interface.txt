### Configuration : User Interface

<span style="color:#4559bf;">_The user interface tab provide settings
for how the user interface is displayed._</span>

___Theme___: The theme to use for the user interface.  Mac OS and
Windows only have a few themes shipped with the installation package.
Other themes may be installed if wanted.

___Font___: The display font used for the main user interface.

___Listing Font___: The font used to display lists of songs in the music
queues, requests, song selection, &etc.

___Accent Color___: The color used for important status messages and for
the color of the buttons.
