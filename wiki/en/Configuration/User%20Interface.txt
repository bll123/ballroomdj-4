### Configure : User Interface

__Theme__: The theme to use for the user interface.  Mac OS and
Windows only have a few themes shipped with the installation package.
Other themes may be installed if wanted.

__Font__: The display font used for the main user interface.

__Listing Font__: The font used to display lists of songs in the music
queues, requests, song selection, &etc.

__Scale__: The scale of the user interface.  Only needs to be changed
for 4K or larger displays.

__Accent Color__: The color used for important status messages and for
the color of the buttons.
