### Configuration : Marquee

<span style="color:#4559bf;">_The marquee can be displayed on a
monitor or projected on to a screen or wall so that the dancers can
see the upcoming dances._</span>

If you do not have a monitor or projector where the marquee can be
displayed, consider using [Announcements](en-Subjects-Announcements)
or using the [Mobile%20Marquee](en-Configuration-Mobile%20Marquee).

___Show Marquee___: Set to _Off_, _Minimized_ or _Visible_.  The marquee
starts when the player is started.

___Marquee Theme___: The interface theme used to display the marquee.

___Marquee Font___: The font used for the marquee.  A narrow font works
best for the marquee.

___Queue Length___: The number of upcoming songs to display on the
marquee.

___Show Song Information___: If enabled, the song artist and title will
be displayed on the marquee.

___Accent Color___: The color to use for the current dance and countdown
timer.
