### Configure : Marquee

If you do not have a monitor or projector where the marquee can be
displayed, consider using [Announcements](en-Subjects-Announcements)
or using the [Mobile%20Marquee](en-Configuration-Mobile%20Marquee).

__Show Marquee__: Set to _Off_, _Minimized_ or _Visible_.  The marquee
starts when the player is started.

__Marquee Theme__: The interface theme used to display the marquee.

__Marquee Font__: The font used for the marquee.  A narrow font works
best for the marquee.

__Queue Length__: The number of upcoming songs to display on the
marquee.

__Show Song Information__: If enabled, the song information configured
in
[Configure&nbsp;/&nbsp;Display&nbsp;Settings](en-Configure-Display%20Settings)
will be displayed on the marquee.  Only three items can be displayed
in the marquee song information display.

__Accent Color__: The color to use for the current dance and countdown
timer.
