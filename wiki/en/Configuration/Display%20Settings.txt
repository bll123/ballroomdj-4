### Configure : Display Settings

The display settings are used to configure which columns are displayed
for the music queues, song selection and song editor.

Each type of listing may be configured individually.

The following displays can be configured:

* Music&nbsp;Queue
* History
* Request
* Song&nbsp;List
* Song&nbsp;Selection
* Side&#8209;by&#8209;Side&nbsp;Song&nbsp;List
* Side&#8209;by&#8209;Side&nbsp;Song&nbsp;Selection
* Music&nbsp;Manager
* Song&nbsp;Editor (three columns)
* Audio&nbsp;ID&nbsp;Match&nbsp;List
* Audio&nbsp;ID
* Marquee (song information)
* Current Song (currently playing song information)

The marquee will only display at most three items in the song
information display.

The player will only display at most five items in the current song
information display.

Use the spinbox at the top of the screen to select which display to
configure.  Select a column and use the left and right
arrows&nbsp;(![right](https://ballroomdj4.sourceforge.io/wikiimg/LeftButton-B.png),&nbsp;![right](https://ballroomdj4.sourceforge.io/wikiimg/RightButton-B.png))
to remove and add columns to display.  Use the up and down
arrows&nbsp;(![move-up](https://ballroomdj4.sourceforge.io/wikiimg/UpButton-B.png),&nbsp;![move-down](https://ballroomdj4.sourceforge.io/wikiimg/DownButton-B.png))
to change the order in which the columns are displayed.

Information on each display item can be found here:
