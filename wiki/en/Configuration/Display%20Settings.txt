### Configure : Display Settings

The display settings are used to configure which columns are displayed
for the music queues, song selection and song editor.

Each type of listing may be configured individually.

The following displays can be configured: Music&nbsp;Queue, History,
Requests, Song&nbsp;List, Song&nbsp;Selection,
Side&#8209;by&#8209;Side&nbsp;Song&nbsp;List,
Side&#8209;by&#8209;Side&nbsp;Song&nbsp;Selection, Music&nbsp;Manager,
Song&nbsp;Editor (three columns).

Use the spinbox at the top of the screen to select which display to
configure.  Select a column and use the left and right arrows
(![left](https://ballroomdj4.sourceforge.io/wikiimg/LeftButton-B.png),
![right](https://ballroomdj4.sourceforge.io/wikiimg/RightButton-B.png))
to remove and add columns to display.  Use the up and down arrows
(![move-up](https://ballroomdj4.sourceforge.io/wikiimg/UpButton-B.png),
![move-down](https://ballroomdj4.sourceforge.io/wikiimg/DownButton-B.png))
to change the order in which the columns are displayed.
