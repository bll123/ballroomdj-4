### Configuration : Display Settings

The display settings are used to configure which columns are displayed
for the music queues, song selection and song editor.

Each type of listing may be configured individually.

The following displays can be configured: Music&nbsp;Queue,
History, Requests, Song&nbsp;List, Song&nbsp;Selection, Easy&nbsp;Song&nbsp;List,
Easy&nbsp;Song&nbsp;Selection, Music&nbsp;Manager, Song Editor (three columns).

Use the spinbox at the top of the screen to select which display to
configure.  Select a column and use the left and right arrows to add
and remove columns to display.  Use the up and down arrows to change
the order in which the columns are displayed.
