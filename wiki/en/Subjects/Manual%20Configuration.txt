### Manual Configuration Items

These items are not configurable via the user interface.

###### data/bdjconfig.txt

 * CLOCKDISP : Clock Display
    * local
      standard display
    * iso
      day-of-week yyyy-mm-dd hh:mm
    * time12
      hh:mm am/pm
    * time24
      hh:mm
    * off

###### data/profile00/bdjconfig.txt

  * FADETYPE : Fade Type
    triangle (blue), quartersine (red), halfsine (green),
    exponentialsine (cyan), invertedparaboloa (black), quadratic (magenta)
  * MQ_INFO_COL
    Marquee Artist/Title Line Color.
  * MQ_TEXT_COL
    Marquee Text Color.
  * UI_ERROR_COL
    User Interface Error Color.
    The color of error messages.
  * UI_MARK_COL
    User Interface Mark Color.
    The color for the request list in-queue mark.
