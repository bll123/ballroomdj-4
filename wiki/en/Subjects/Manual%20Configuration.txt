### Manual Configuration Items

These items are not configurable via the user interface.

###### data/bdjconfig.txt

 * CLOCKDISP : Clock Display
    * local
      standard display
    * iso
      day-of-week yyyy-mm-dd hh:mm
    * time12
      hh:mm am/pm
    * time24
      hh:mm
    * off

###### data/profile00/bdjconfig.txt

  * FADETYPE : Fade Type
    triangle (blue), quartersine (red), halfsine (green),
    exponentialsine (cyan), invertedparabola (black), quadratic (magenta)
<div markdown style="margin-right: 20px; text-align: center;">
![fadeimg-D](https://ballroomdj4.sourceforge.io/wikiimg/en/Subjects/fadeimg-D.png)
<br><span style="color:#4559bf;">__Fade Types__</span> </div>
  * MQ_INFO_SEP
    Separator character(s) for the marquee information display.
  * MQ_INFO_COL
    Marquee Artist/Title Line Color.
    Must be specified as '#rrggbb'.
  * MQ_TEXT_COL
    Marquee Text Color.
    Must be specified as '#rrggbb'.
  * PLAYER_UI_SEP
    Separator character(s) for the player currently-playing display.
  * UI_ERROR_COL
    User Interface Error Color.
    The color of error messages.
    Must be specified as '#rrggbb'.
  * UI_MARK_COL
    User Interface Mark Color.
    The color for the request list in-queue mark.
    Must be specified as '#rrggbb'.
  * UI_MARK_TEXT
    The character(s) to use to display the mark.
  * UI_ROWSEL_COL
    Row selection color.

###### data/favorites.txt

The favorites.txt file has internal documentation specifying what can
be changed.  All of the display characters may be changed.

###### data/orgopt.txt

The orgopt.txt file has internal documentation.
[Contact](en-Getting%20Help) the developers for more information.

###### data/sortopt.txt

The sortopt.txt file has the list of selections for sorting the
display. [Contact](en-Getting%20Help) the developers for more
information.
