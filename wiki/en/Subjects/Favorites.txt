### Favorites

##### Settings up Favorites for Use

In
[Configuration&nbsp;/&nbsp;Display&nbsp;Settings](en-Configuration-Display%20Settings),
add the _Favorite_ selection to the _Music&nbsp;Queue_, _History_,
_Request_, _Song&nbsp;List_, _Song&nbsp;Selection_,
_Side-by-Side&nbsp;Song&nbsp;List_,
_Side-by-Side&nbsp;Song&nbsp;Selection_, _Music&nbsp;Manager_ and to
one of the _Song&nbsp;Editor_ columns.  Be sure to move the _Favorite_
selection to the bottom of the list on the right.

These display settings can be adjusted to meet your needs, but for the
purposes of this example, the _Favorite_ selection is added everywhere.
The _Favorite_ column that will be displayed is narrow and does not
take up much space.

##### Using Favorites

To change the favorite selection, it is not necessary to use the
[Song&nbsp;Editor](en-Management-Song%20Editor).  The favorite
selection can be changed anywhere it is displayed, including in the
[Player](en-Player-Player). Simply click with the mouse button on the favorite symbol for
the song and it will change to the next favorite symbol/color.
Continue clicking on the favorite symbol until it is set to your
choice.  The change to the favorite selection will automatically be
saved to the database.

##### Filtering the Song Selection

The song selection can be filtered by favorites.  Using the song
selection filter, select the favorite selection you want
to view and select _Apply_.

##### Configuring the Favorite Display

At this time, there is no easy method to configure the symbols and colors.
To add or change the symbols and colors, the favorite configuration file must
[edited manually](en-Subjects-Manual%20Configuration).

If there is enough interest in being able to easily change the symbols
and colors, I can add a configuration screen for that.
