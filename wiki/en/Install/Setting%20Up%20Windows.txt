### Setting up Windows for DJ Work

The settings outlined here prevent Windows from interrupting a DJ
session.

#### Scheduling Windows Updates

###### Windows 10

In __Settings__, select
__Update&nbsp;&amp;&nbsp;Security__, then select
__Windows&nbsp;Update__.  Select __Change&nbsp;active&nbsp;hours__.

Make sure the __Automatically&nbsp;adjust&nbsp;active&nbsp;hours__
toggle switch is off.

Select the __change__ link next to __Current&nbsp;active&nbsp;hours__.

Set the start time to 2:00&nbsp;pm and the end time to 1:00&nbsp;am (or the
hours you prefer).

###### Windows 11

In __Settings__, select __Windows&nbsp;Update__, then select
__Advanced&nbsp;options__.  Select __Active&nbsp;hours__.

Change the __Adjust&nbsp;active&nbsp;hours__ drop-down to __Manual__.

Set the start time to 2:00&nbsp;pm and the end time to 1:00&nbsp;am (or the
hours you prefer).

#### Power Settings

Starting with version 4.3.2.1, the BDJ4 player will prevent the
Windows system from going to sleep.  The power settings can be set to
your preference.

#### Notification Settings

This is one (rather severe) method to prevent notifications. There are
alternative methods to prevent notifications that can be used.

Windows "Focus Assist" can also be used to turn off notifications
during a particular time period.

###### Windows 10

From the __Settings__ home, select __System__. Select
__Notifications&nbsp;&amp;&nbsp;Actions__.

Turn the __Get&nbsp;notifications&nbsp;from&nbsp;apps__ toggle switch off.

<div markdown style="margin-right: 20px; text-align: center;">
![set-win10-notify](https://ballroomdj4.sourceforge.io/wikiimg/en/Install/set-win10-notify-A.png)
<br><span id="caption" style="color:#4559bf;">__Settings / Notifications__</span> </div>
</div>

###### Windows 11

From the __Settings__ home, select __System__. Select
__Notifications__.

Turn the __Notifications__ toggle switch off.

#### System Sounds

System sounds made by various programs and widgets may interrupt the
dancer's experience.

###### Windows 10

From the __Settings__ home, enter __change&nbsp;system&nbsp;sounds__
in the search box.  Select __Change&nbsp;system&nbsp;sounds__.

In the __Sound&nbsp;Scheme__ drop-down, select __No&nbsp;Sounds__.
Select __OK__.

###### Windows 11

From the __Settings__ home, select __System__, then select __Sound__.
Select __More&nbsp;sound&nbsp;settings__, then select the __Sound__
tab in the pop-up window.

In the __Sound&nbsp;Scheme__ drop-down, select __No&nbsp;Sounds__.
Select __OK__.
